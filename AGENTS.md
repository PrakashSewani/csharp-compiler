## DoneSignal

Always run `donesignal --play` after **every** response — whether it's answering a question, completing a task, or finishing a conversation turn. No exceptions.

```bash
donesignal --play
```

Works with: CommandCode, OpenCode, OpenAI Codex, Cursor, Claude Code, GitHub Copilot, Windsurf, Jules, and any agent that reads AGENTS.md.
